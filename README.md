# CV Personnel Interactif

## 👩‍💻 Projet
Ce projet est un CV interactif développé dans le cadre du module **Programmation Web 2**.

---

## 🚀 Technologies utilisées
- HTML5 / CSS3
- JavaScript (ES6)
- jQuery
- ReactJS (CDN)
- GitHub Pages

---

## ✨ Fonctionnalités
- CV responsive et moderne
- Animation des compétences (jQuery)
- Accordéon pour la formation
- Composants React (ProjectCard + SkillCard)
- Validation de formulaire
- Design professionnel rose

---

## 📂 Structure du projet
- index.html
- css/style.css
- js/main.js
- js/components.js
